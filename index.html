시각적인 완성도뿐 아니라 사용자 경험의 본질을 이해하고 문제를 해결하는 디자이너로 성장하고 싶습니다.

UI/UX Designer & Web Developer

저는 사용자가 ‘왜 이 경험을 하게 되는가’를 끝까지 탐구하는 디자이너입니다.
구조와 패턴을 관찰하며 사용성과 의미를 고민해왔고, 게임 그래픽·UX/UI 교육과 실무 경험을 통해 시각적 표현과 사용자 흐름을 함께 바라보는 사고방식을 쌓았습니다.

Figma 기반 UI 디자인부터 HTML, CSS, JavaScript 퍼블리싱까지 전 과정을 직접 경험하며 프로젝트를 진행해왔습니다.
디자인과 개발 사이의 간극을 줄이는 역할을 지향합니다.

문제를 발견하고 해결 방향을 고민하는 데 주저함이 없으며, 책임감 있게 작업을 마무리하기 위해 꼼꼼한 검수와 지속적인 자기 성장을 중요하게 생각합니다. 사용자의 감정과 관점을 중심에 두고, 더 나은 경험을 만들기 위해 깊이 탐구하고 끊임없이 개선하려 노력하는 디자이너입니다.
